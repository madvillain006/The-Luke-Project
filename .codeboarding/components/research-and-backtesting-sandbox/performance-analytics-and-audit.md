# Performance Analytics & Audit

## Component Description

Provides the statistical feedback loop by analyzing simulation outcomes. It audits "coverage" (how many signals were captured) and "alignment" (how signals performed relative to heatmap confluences), producing win-rate metrics that drive system optimization.
